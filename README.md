# Blog App

A full stack blog application where users can create, edit, and delete blog posts with image uploads.


## Features
- User authentication (signup, login, logout)
- Create, edit, delete blog posts
- Featured image upload
- Rich text editor
- Protected routes
- Active/inactive post status

## Tech Stack
- **Frontend:** React, Redux Toolkit, Tailwind CSS
- **Backend:** Appwrite
- **Other:** React Router, React Hook Form, TinyMCE
